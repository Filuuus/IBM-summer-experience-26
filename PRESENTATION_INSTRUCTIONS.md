# Instructions to Create PowerPoint Presentation

## Option 1: Using the Python Script (Recommended)

If you have Python installed on your system:

1. **Install python-pptx library**:
   ```bash
   pip install python-pptx
   ```

2. **Run the script**:
   ```bash
   python create_presentation.py
   ```

3. **Output**: `CropAnalytics_Client_Presentation.pptx` will be created in the current directory

---

## Option 2: Manual Creation from Markdown

Use the markdown file `CropAnalytics_Client_Presentation.md` as a guide:

1. Open PowerPoint
2. Create 15 slides following the markdown structure
3. Copy content from each section
4. Apply professional theme and formatting

---

## Option 3: Online Converter

Use an online markdown-to-PowerPoint converter:

1. Visit: https://www.markdowntoppt.com/ or similar service
2. Upload `CropAnalytics_Client_Presentation.md`
3. Download the generated PPTX file

## Quick PowerShell Command to Install and Run

If you have Python in your PATH:

```powershell
python -m pip install python-pptx; python create_presentation.py
```

Or if Python is in a specific location:

```powershell
C:\Python311\python.exe -m pip install python-pptx
C:\Python311\python.exe create_presentation.py
```

---

## Troubleshooting

**If Python is not found:**
- Install Python from https://www.python.org/downloads/
- Or use Windows Store to install Python
- Make sure to check "Add Python to PATH" during installation

**If pip is not found:**
- Use: `python -m ensurepip --upgrade`
- Then: `python -m pip install python-pptx`
