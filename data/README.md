# Carpeta de Datos Locales

Coloca aquí los 7 archivos CSV iniciales para poblar la base de datos local.
Los archivos necesarios son:
- Ciclos.csv
- Estados.csv
- Hibridos.csv
- Laboratorio.csv
- Municipios.csv
- OpenMeteo.csv
- Terrenos.csv

**Nota:** Estos archivos están ignorados por Git por seguridad. Ejecuta `python manage.py cargar_datos` después de colocarlos aquí.
